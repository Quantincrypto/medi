### 12. Page Speed Signals
**Status:** ⚠️ WEAK

- Site is a React SPA (client-side rendered via Vite)
- **Risk:** Initial render requires JavaScript execution — may impact Core Web Vitals (LCP, FID, CLS) on slow connections
- No server-side rendering (SSR) or static site generation (SSG) detected
- **Fix options:**
  1. Add Vite SSR or switch to Next.js/Astro for SSG on key landing pages
  2. Implement code-splitting and lazy loading for page components (check `vite.config.ts`)
  3. Pre-render critical pages using a service like Prerender.io
- **[ACTION NEEDED]:** Run Google PageSpeed Insights and Lighthouse audit on live site to get actual scores

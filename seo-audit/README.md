## [ACTION NEEDED] — Consolidated Client Input Items

The following items require information only the site owner can provide. They are **blocking items** for Phase 4 (Content Creation).

### Schema & Technical
- [ ] Physical business address for LocalBusiness schema (street, city, country)
- [ ] Social media profile URLs for `sameAs` in Organization schema (LinkedIn, Facebook, Twitter/X)
- [ ] Confirm `/logo.png` exists at root domain
- [ ] Confirm `/og-image.jpg` exists and is 1200×630px
- [ ] Confirm search URL pattern for SearchAction (e.g. `/catalogue?q={search_term_string}`)

### Kenya Content
- [ ] Names of Kenyan hospital/clinic customers (for testimonials — can be anonymised)
- [ ] PPB import permit fee and processing time for medical devices
- [ ] Delivery lead times to Nairobi vs upcountry
- [ ] Kenya-based biomedical engineer partners for local installation

### Nigeria Content
- [ ] Nigerian hospital customer names or quotes
- [ ] NAFDAC device registration fee and typical processing timeline
- [ ] Lagos port customs clearance agent contact (if vetted partner exists)
- [ ] Active state government hospital procurement programs

### Refurbished Hub Page
- [ ] Price ranges for refurbished vs new (CT, MRI, Ultrasound, X-Ray)
- [ ] Direct engagement with any African regulatory body — letters or approvals
- [ ] Downloadable supplier checklist/datasheet for lead magnet
- [ ] NGO or government tender wins to reference (even anonymised)

### Kenya Import Guide
- [ ] PPB device classification for refurbished imaging equipment (Class A/B/C/D)
- [ ] Current KRA import duty rate and VAT exemption status for hospitals
- [ ] Recommended Kenya customs broker / clearing agent

### Ultrasound Category Page
- [ ] Price ranges for portable POCUS vs full cart-based ultrasound
- [ ] Ultrasound brands carried by verified suppliers (GE, Siemens, Philips, Mindray, SonoScape?)
- [ ] Whether application training is included in standard procurement packages
